# Manipulation de Polynômes

Projet Java de manipulation de polynômes d'une variable.

## Structure
```
polynomes/
├── src/
│   └── polynomes/
│       ├── Monome.java
│       ├── Polynome.java
│       ├── Analyseur.java
│       ├── Operations.java
│       ├── GarbageCollector.java
│       └── Main.java
├── out/
├── run.sh
└── README.md
```

## Compiler et exécuter
```bash
./run.sh
```
